**Key Technical Concepts**
- **Hybrid Storage Strategy**: An abstraction layer () enables switching between local file storage for development and AWS S3 for production.
- **Per-Tenant Encryption**: Unique encryption keys are derived for each estate using a master key and a per-estate salt, ensuring strong data isolation.
- **Lazy Data Migration**: A backward-compatibility mechanism automatically upgrades older data to the new encryption standard upon access, avoiding a risky, one-off migration script.
- **AI Persona & Guardrails**: System prompts are used to strictly define the AI's expert scope and behavior, forcing it to politely deflect irrelevant questions.
